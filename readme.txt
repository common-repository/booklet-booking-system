A hotel reservation confirm (by user e-mail) plugin for WordPress sites

== Description ==

Booklet Booking System is a very simple booking system plugin. With this plugin, you add a simple booking form for or anything to reserve for your customers. It has very nice option panel to add a title, subtitle, and email for the form. It can be used in your sites post and pages with very simple WordPress shortcode. Hotel reservation confirm by user e-mail address.

== Installation ==

1. Upload plugin directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Booklet Booking System' menu in WordPress.
3. Go to Settings => Booklet Settings => and follow the options how you need.
